第一步：打开 Chrome 扩展页面

首先，我们需要进入 Chrome 的扩展管理页面。

📋 请复制下面的地址，在 Chrome 地址栏粘贴并回车

chrome://extensions/

✓ 已复制到剪贴板！

第二步：开启开发者模式

这是最关键的一步！必须先开启开发者模式，才能加载本地扩展。

⚠️ 注意

请看页面右上角，找到「开发者模式」开关，点击它变成蓝色开启状态

第三步：选择扩展目录

点击「加载已解压的扩展程序」按钮，然后在弹出的窗口中选择下面的文件夹：

{{EXTENSION_PATH}}

⚠️ 重要提示

  • 必须选择 extension-new 文件夹本身
  • 不要选择文件夹里面的文件
  • 选择后点击「选择文件夹」按钮

第四步：固定扩展并连接服务器

📌 第一步：把扩展固定到工具栏

刚安装的扩展可能在「拼图图标」里面，建议先固定到工具栏：

  • 点击 Chrome 右上角的 拼图图标 🧩
  • 找到 CDP Tunnel
  • 点击旁边的图钉图标 📌 固定到工具栏

🔌 第二步：连接服务器

  • 点击工具栏上的 CDP Tunnel 图标
  • 会弹出配置窗口
  • 确认地址是：ws://localhost:9221/plugin
  • 点击「保存并连接」按钮
🎉

安装完成！

现在回到终端（命令行窗口），运行下面的命令验证连接：

cdp-tunnel status

✅ 验证成功标志

如果看到 「扩展: 已连接」，恭喜你，全部完成了！
