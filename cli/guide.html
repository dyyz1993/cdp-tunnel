🔌 CDP Tunnel 安装指南

Chrome 扩展连接教程（小白专用）

1
打开 Chrome 扩展页面
✓
在 Chrome 地址栏输入 chrome://extensions/ 并回车
✓
或者点击 Chrome 右上角 ⋮ → 更多工具 → 扩展程序

💡 提示

打开后你会看到类似下面的页面：

扩展程序
开发者模式
🔌
CDP Tunnel
Chrome DevTools Protocol Bridge
ID: bchclccgjmihieacfmaelkpfjlghhoph
2
开启开发者模式

⚠️ 重要！必须先开启开发者模式

  • 看页面右上角 → 找到「开发者模式」开关
  • 点击开关，让它变成 蓝色/开启 状态
  • 开启后，页面上方会出现「加载已解压的扩展程序」按钮
扩展程序
开发者模式
← 点击这里
3
选择扩展目录

📁 选择正确的文件夹

  • 点击「加载已解压的扩展程序」按钮
  • 会弹出一个文件选择窗口
  • 找到并选择下面的路径（不要选错！）
扩展目录路径：
/Users/xuyingzhou/Project/study-web/cdp-tunnel2/extension-new

⚠️ 注意

  • 必须选择 extension-new 文件夹本身，而不是里面的文件
  • 选择后点击「选择文件夹」按钮
  • 如果提示错误，检查是否选对了文件夹
4
连接到服务器
1
安装成功后，在 Chrome 右上角找到 🔌 图标（可能需要点击拼图图标才能看到）
2
点击 🔌 图标，会弹出配置窗口
3
确认地址是：ws://localhost:9221/plugin
4
点击「保存并连接」按钮
5
验证连接成功

✅ 检查是否成功

回到终端（命令行窗口），运行：

cdp-tunnel status

如果看到 「扩展: 已连接」，恭喜你，成功了！
