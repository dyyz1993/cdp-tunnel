第一步：打开 Chrome 扩展页面

我们需要先打开 Chrome 的扩展管理页面。请按照下面的方法操作：

💡 两种打开方式

方式一：在地址栏输入 chrome://extensions/ 并回车

方式二：点击 Chrome 右上角 ⋮ → 更多工具 → 扩展程序

扩展程序
开发者模式
打开后你会看到这个页面

第二步：开启开发者模式

这是最关键的一步！必须先开启开发者模式，才能加载我们的扩展。

⚠️ 注意

请看页面右上角，找到「开发者模式」开关，点击它变成蓝色开启状态

扩展程序
开发者模式
👆 点击这里
开启后会出现「加载已解压的扩展程序」按钮

第三步：选择扩展目录

点击「加载已解压的扩展程序」按钮，然后选择下面的文件夹：

/Users/xuyingzhou/Project/study-web/cdp-tunnel2/extension-new

⚠️ 重要提示

必须选择 extension-new 这个文件夹本身，不要选里面的文件！

选择后点击「选择文件夹」按钮

🔌
CDP Tunnel
Chrome DevTools Protocol Bridge

第四步：连接到服务器

扩展安装成功了！现在需要连接到你的服务器。

🔌 连接步骤

1. 在 Chrome 右上角找到 🔌 图标（可能需要点击拼图图标展开）

2. 点击 🔌 图标，会弹出配置窗口

3. 确认地址是：ws://localhost:9221/plugin

4. 点击「保存并连接」按钮

🔌
点击这个图标
🎉

安装完成！

现在回到终端，运行下面的命令验证连接：

cdp-tunnel status

如果看到 「扩展: 已连接」，恭喜你，全部完成了！
